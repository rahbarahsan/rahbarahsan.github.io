Journals

  • Network Intrusion Detection Using Machine Learning Approaches: Addressing Data Imbalance (Accepted in IET Cyber-Physical Systems: Theory & Applications).

    • A comparative analysis of different resampling techniques is shown with seven machine learning algorithms (Decision Tree, Random Forest, Extra Tree, Logistic Regression L1 & L2 penalty, Naïve Bayes, SVM) and two deep learning algorithms (Fully Connected Neural Network (FCN) and Autoencoder with FCN) in this research. A voting classifier is also proposed to ensemble these machine learning algorithms to achieve better performance.

  • GAN-based Oversampling for Anomaly Detection (Submitted in IET Cyber-Physical Systems: Theory & Applications).

    • This research aims to improve the network intrusion detection system’s performance using an advanced level oversampling technique such as Conditional Generative Adversarial Network (CGAN). Also, a hybrid solution is proposed involving CGAN and Adversarial Environment Reinforcement Learning (AE-RL). This solution tries to improve the performance of an algorithm-level approach (AE-RL) by combining it with a data-level approach (CGAN).

    Conferences

  • Prediction of Autism Severity Level in Bangladesh Using Fuzzy Logic: FIS and ANFIS (Published in MISSI 2018 11th conference on Multimedia and Network Information Systems):

    • This research focuses on attaining the severity level of autism using fuzzy methods like MAMFIS and ANFIS. A comparative study of FIS and ANFIS has also been shown.

  • Analysis of soil properties and climate data of Bangladesh to predict crop yields by applying data mining techniques (Published in 17th IEEE/ACIS International Conference on Computer and Information Science):

    • In this research, different types of data mining techniques like K-means, PAM, CLARA, and DBSCAN are applied for clustering, and four regression models are used to predict crop yields.

  • EZ Parking: Smart Parking Reservation using Internet of Things (Published in ICACSIS 2018 the 10th International Conference on Advanced Computer Science and Information Systems):

    • An Android application had been made where the user can give their parking place for rent, or the user can also search for a free parking spot. Parking reservations can be completed entirely online, and a device also set in the parking space will notify the park space owner of the parking spot’s availability.

    Dissertations

  • A Detailed Analysis of The Multi-class Classification Problem in Network Intrusion Detection using Resampling Techniques (Masters Thesis)

    • A Detailed Analysis of The Multi-class Classification Problem in Network Intrusion Detection using Resampling Techniques (Masters Thesis) This dissertation aims to provide a detailed analysis of the resampling techniques in the network intrusion domain. It will help researchers to understand the potential of resampling techniques and how to utilize them to solve the imbalanced dataset problem of the network intrusion detection domain.

  • An Embedded, Scalable, Cloud-Based A.I. System for Answering Factoid Questions Printed in Bangla (Undergraduate Thesis).

    • This research was related to Natural Language Processing, which developed an embedded Artificial Intelligence system for answering factoid questions printed in the Bengali language. This system takes an image of a Bengali printed question as an input and can infer an answer in Bengali using services like Google Vision, Microsoft Translator, and Wolfram Alpha.
